Página web html con scripts en javascript. Se emplea jQuery para soporte multinavegador y la API de Google Maps, así como CSS para presentarla. Ilustra el uso de iframes, eventos de javascript, almacenamiento de variables locales e interacción con jQuery.

La página consta de:
-Un video explicativo.
-Un iframe con Google Maps donde se muestra nuestra posición actual(Roja) y la de la última vez que accedimos a la página(Verde) desde este navegador.
-Dos formularios para modificar la posición anterior o la actual.
